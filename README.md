# mlp-regressor-01
Treinamento de uma rede neural MLP aplicado a um problema real de regressão: a previsão do valor mediano de casas na Califórnia. O projeto realiza divisão dos dados pelo método Hold-out, normalização, treinamento do modelo, geração de previsões e avaliação com métricas clássicas de regressão — MAE, MSE, RMSE, MAPE e R².
